# 💳 Banking Management System

A web-based Banking Management System developed using Django. It allows users to manage bank accounts and perform basic banking operations.

---

## 🚀 Features
- User Registration & Login  
- Account Management  
- Deposit & Withdraw Money  
- Transaction Handling  
- Admin Panel  

---

## 🛠️ Tech Stack
- Python  
- Django  
- SQLite  
- HTML, CSS  

---

## ⚙️ How to Run

1. Clone the repository:
git clone https://github.com/nikhil32354/Banking_System.git

2. Go to project folder:
cd Banking_System

3. Install Django:
pip install django

4. Run migrations:
python manage.py migrate

5. Start server:
python manage.py runserver

6. Open browser:
http://127.0.0.1:8000/
